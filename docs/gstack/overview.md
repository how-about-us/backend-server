# 제품 기획 개요

> 이 문서는 /office-hours 세션에서 도출된 제품 레벨 맥락을 누적 관리합니다.

## 서비스 개요

**"우리어때"** — 여행 계획을 함께 세우는 협업 웹 서비스.
- 우측 지도 /좌측 채팅 + 일정 + 북마크
- AI 에이전트가 채팅방 팀원으로 참여하여 여행 계획 수립 보조

## 핵심 문제
여행 계획을 세울 때 도구가 분산됨 (카카오톡 채팅 + 엑셀 일정 + URL 장소 공유).
올인원 플랫폼으로 도구 분산을 해소하되, 단순히 합치는 것만으로는 부족함.
협업 참여를 유도하는 것이 핵심.

## 타겟 사용자
해외/장기 여행을 구체적으로 계획하는 그룹 (3-8명). 
여행계획을 세우는 과정에서 협업이 필요한 사용자.

> 아직 구체적 사용자 인터뷰/검증은 미실시 (pre-product 단계).

## 차별점
1. **올인원**: 지도 + 채팅 + 일정 + 보관함이 한 화면
2. **AI 팀원**: 채팅에서 AI가 여행 계획 수립을 보조
3. **실시간 협업**: WebSocket 기반 접속 상태, 변경 브로드캐스트

## 제품 단계
- Pre-product (아이디어 단계, 개발 진행 중)
- 수요 증거: 본인 경험 기반
