# FE-001-03: 로그아웃

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 저장소: Redis (`refresh:{userId}`)

## 개요

Redis에서 Refresh Token을 삭제해 토큰을 무효화한다. Access Token은 서버에서 강제 취소 불가하므로 짧은 만료 시간(15분)에 의존.

---

## API 명세

### `POST /auth/logout`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

---

## 구현 노트

- JWT에서 `userId` 추출 → Redis `refresh:{userId}` 키 삭제
- Access Token 블랙리스트는 MVP에서 미적용 (만료 시간 짧으므로 허용)
- 인증 필터를 통과한 요청만 처리 (유효한 Access Token 필요)
