# FE-001-04: 내 정보 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `users`

## 개요

현재 로그인된 사용자의 프로필 정보를 반환한다.

---

## API 명세

### `GET /users/me`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
{
  "id": 1,
  "email": "user@example.com",
  "name": "홍길동",
  "profileImage": "https://..."
}
```

---

## 구현 노트

- `@AuthenticationPrincipal`로 JWT에서 추출한 `userId`를 주입
- `users` 테이블에서 단건 조회 (`findById`)
- 탈퇴/미존재 사용자 요청 시 `404` 반환

---

## DoD

- [ ] 유효한 Access Token으로 본인 프로필이 반환된다
- [ ] 타인 정보는 이 API로 조회 불가하다
