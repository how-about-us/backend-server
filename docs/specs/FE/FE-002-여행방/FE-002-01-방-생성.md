# FE-002-01: 방 생성

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `rooms`, `room_members`

## 개요

여행 방을 생성한다. 생성자는 자동으로 ADMIN 역할로 `room_members`에 등록되며, `invite_code`가 발급된다.

---

## API 명세

### `POST /rooms`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "title": "제주도 여행",
  "destination": "제주도",
  "startDate": "2025-08-01",
  "endDate": "2025-08-05"
}
```

**Response `201`**
```json
{
  "id": 1,
  "title": "제주도 여행",
  "destination": "제주도",
  "startDate": "2025-08-01",
  "endDate": "2025-08-05",
  "inviteCode": "abc123",
  "memberCount": 1
}
```

---

## 구현 노트

- `invite_code`: UUID 앞 8자리 또는 별도 랜덤 문자열, UNIQUE 제약
- 방 생성과 동시에 `room_members`에 `role=ADMIN`으로 생성자 등록 → 단일 트랜잭션
- `startDate` ≤ `endDate` 유효성 검증
