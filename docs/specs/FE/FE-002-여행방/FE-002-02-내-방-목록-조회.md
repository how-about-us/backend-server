# FE-002-02: 내 방 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `rooms`, `room_members`

## 개요

현재 로그인 사용자가 참여 중인 방 목록을 반환한다.

---

## API 명세

### `GET /rooms`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
[
  {
    "id": 1,
    "title": "제주도 여행",
    "destination": "제주도",
    "startDate": "2025-08-01",
    "endDate": "2025-08-05",
    "memberCount": 3,
    "myRole": "ADMIN"
  }
]
```

---

## 구현 노트

- `room_members.user_id = :userId` 조건으로 JOIN 쿼리
- N+1 방지: JPQL fetch join 또는 `@EntityGraph`로 `rooms` 한 번에 조회
- 최신 방 순으로 정렬 (`rooms.created_at DESC`)
