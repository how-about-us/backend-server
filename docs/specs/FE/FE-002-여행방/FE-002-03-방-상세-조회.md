# FE-002-03: 방 상세 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `rooms`, `room_members`

## 개요

방의 메타정보(제목, 여행지, 날짜, 멤버 수 등)를 반환한다.

---

## API 명세

### `GET /rooms/{roomId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
{
  "id": 1,
  "title": "제주도 여행",
  "destination": "제주도",
  "startDate": "2025-08-01",
  "endDate": "2025-08-05",
  "inviteCode": "abc123",
  "memberCount": 3,
  "myRole": "ADMIN"
}
```

**Error Cases**
- `403` — 방 미참여자 접근
- `404` — 존재하지 않는 방

---

## 구현 노트

- 요청자가 `room_members`에 속해 있는지 확인 후 응답 (권한 검증)
- `memberCount`는 서브쿼리 또는 별도 count 쿼리로 조회
