# FE-002-04: 방 수정

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `rooms`

## 개요

방 제목, 여행지, 날짜를 수정한다. ADMIN만 가능.

---

## API 명세

### `PATCH /rooms/{roomId}`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body** (변경할 필드만 전송)
```json
{
  "title": "제주도 여행 v2",
  "destination": "제주 서귀포",
  "startDate": "2025-08-02",
  "endDate": "2025-08-06"
}
```

**Response `200`** — 수정된 방 상세 정보

**Error Cases**
- `403` — ADMIN이 아닌 경우
- `404` — 존재하지 않는 방

---

## 구현 노트

- `room_members.role = ADMIN` 검증 후 수정
- 전달된 필드만 업데이트 (null 필드는 기존값 유지)
- `startDate` ≤ `endDate` 유효성 검증

---
