# FE-002-06: 초대 코드 재발급

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `rooms`

## 개요

기존 `invite_code`를 폐기하고 새 코드를 발급한다. ADMIN만 가능.

---

## API 명세

### `POST /rooms/{roomId}/invite-code`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
{
  "inviteCode": "xyz789"
}
```

**Error Cases**
- `403` — ADMIN이 아닌 경우

---

## 구현 노트

- 새 UUID 기반 코드 생성 → `rooms.invite_code` 업데이트
- 이전 코드로 입장 시도 시 자동으로 `404` 처리됨 (코드 교체로 무효화)
