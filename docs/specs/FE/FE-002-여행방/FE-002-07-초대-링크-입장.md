# FE-002-07: 초대 링크로 방 입장

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `rooms`, `room_members`

## 개요

`invite_code`를 통해 방에 입장한다. 이미 참여 중인 경우 중복 등록 없이 정상 응답.

---

## API 명세

### `POST /rooms/join`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "inviteCode": "abc123"
}
```

**Response `200`**
```json
{
  "roomId": 1,
  "title": "제주도 여행"
}
```

**Error Cases**
- `404` — 유효하지 않은 초대 코드

---

## 구현 노트

- `rooms.invite_code`로 방 조회 → 없으면 `404`
- 이미 `room_members`에 존재하면 `200`으로 방 정보만 반환 (멱등 처리)
- 신규 입장 시 `role=MEMBER`로 `room_members` 삽입
