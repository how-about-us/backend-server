# FE-003-01: 방 멤버 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `room_members`, `users`, Redis (`connected_users`)

## 개요

방 참여자 목록과 각 멤버의 역할 및 실시간 접속 상태를 반환한다.

---

## API 명세

### `GET /rooms/{roomId}/members`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
[
  {
    "userId": 1,
    "name": "홍길동",
    "profileImage": "https://...",
    "role": "ADMIN",
    "online": true
  },
  {
    "userId": 2,
    "name": "김철수",
    "profileImage": "https://...",
    "role": "MEMBER",
    "online": false
  }
]
```

---

## 구현 노트

- `room_members` JOIN `users` 쿼리로 멤버 목록 조회
- `online` 여부: Redis `connected_users:{roomId}` SET에 `userId` 포함 여부로 판단
- 방 비참여자 요청 시 `403`
