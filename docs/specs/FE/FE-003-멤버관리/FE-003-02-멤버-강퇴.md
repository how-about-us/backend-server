# FE-003-02: 멤버 강퇴

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `room_members`

## 개요

ADMIN이 특정 멤버를 방에서 내보낸다.

---

## API 명세

### `DELETE /rooms/{roomId}/members/{targetUserId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

**Error Cases**
- `403` — 요청자가 ADMIN이 아님
- `400` — ADMIN이 자기 자신을 강퇴하려는 경우
- `404` — 대상 멤버가 해당 방에 없음

---

## 구현 노트

- 요청자 `role = ADMIN` 검증
- 자기 자신 강퇴 시 `400` (방장은 방 나가기 또는 위임 후 퇴장)
- 강퇴 후 해당 유저의 WebSocket 세션에 강퇴 이벤트 발행 (WS 연결 끊기)

---

## DoD

- [ ] ADMIN만 멤버 강퇴가 가능하다
- [ ] 강퇴된 멤버는 `room_members`에서 제거된다
- [ ] ADMIN 자기 자신 강퇴 시 `400` 반환된다
