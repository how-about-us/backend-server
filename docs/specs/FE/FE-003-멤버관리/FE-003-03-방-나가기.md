## 개요

본인이 방에서 탈퇴한다.

---

## API 명세

### `DELETE /rooms/{roomId}/members/me`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

**Error Cases**
- `400` — ADMIN이 나가려는 경우 (멤버가 남아 있을 때)
- `404` — 해당 방의 멤버가 아닌 경우

---

## 구현 노트

- ADMIN이 마지막 멤버인 경우: 방도 함께 삭제 (또는 정책 결정 필요)
- ADMIN이 나가려는데 다른 멤버가 있으면 `400` → 방장 위임 먼저 요구
- 나가기 후 Redis `connected_users:{roomId}`에서 userId 제거
