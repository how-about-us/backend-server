## 개요

WebSocket으로 현재 방에 접속 중인 유저 목록을 반환한다.

---

## API 명세

### `GET /rooms/{roomId}/members/online`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
{
  "onlineUserIds": [1, 3, 5]
}
```
