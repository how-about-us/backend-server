# FE-004-01: 장소 검색

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `places`
- 외부 API: Google Places API (Text Search)

## 개요

키워드로 장소를 검색한다. 결과를 `places` 테이블에 캐싱(upsert)하고 반환한다.

---

## API 명세

### `GET /places/search?q={keyword}&roomId={roomId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
[
  {
    "placeId": "ChIJ...",
    "name": "성산일출봉",
    "address": "제주특별자치도 서귀포시 성산읍",
    "latitude": 33.458,
    "longitude": 126.942,
    "rating": 4.7,
    "photoUrl": "https://..."
  }
]
```

---

## 구현 노트

- Google Places Text Search API 호출 → 결과를 `places` 테이블에 `google_place_id` 기준 upsert
- `roomId`는 방 참여자 권한 확인용
- API 호출 실패 시 `502` 반환
- 결과 개수: 기본 10건 (Google API 한도 내)
