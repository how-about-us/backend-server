# FE-004-02: 장소 상세 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `places`

## 개요

DB에 캐싱된 장소의 상세 정보를 반환한다.

---

## API 명세

### `GET /places/{placeId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
{
  "placeId": "ChIJ...",
  "name": "성산일출봉",
  "address": "제주특별자치도 서귀포시 성산읍",
  "latitude": 33.458,
  "longitude": 126.942,
  "rating": 4.7,
  "photoUrl": "https://...",
  "openingHours": "09:00 - 20:00"
}
```

**Error Cases**
- `404` — DB에 없는 placeId (검색을 먼저 수행해야 함)

---

## 구현 노트

- `places` 테이블 단건 조회 (`google_place_id` 기준)
- 상세 정보가 부족하면 Google Places Details API 추가 호출 후 upsert 고려
