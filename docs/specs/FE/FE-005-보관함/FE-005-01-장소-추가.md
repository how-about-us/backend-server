# FE-005-01: 보관함에 장소 추가

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `bookmarks`, `places`

## 개요

검색된 장소를 방의 보관함(후보지)에 등록한다.

---

## API 명세

### `POST /rooms/{roomId}/bookmarks`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "placeId": "ChIJ...",
  "category": "ATTRACTION",
  "memo": "꼭 가고 싶은 곳"
}
```

**Response `201`**
```json
{
  "bookmarkId": 10,
  "placeId": "ChIJ...",
  "placeName": "성산일출봉",
  "category": "ATTRACTION",
  "memo": "꼭 가고 싶은 곳"
}
```

**Error Cases**
- `404` — 존재하지 않는 `placeId`
- `409` — 이미 보관함에 등록된 장소

---

## 구현 노트

- `places` 테이블에 해당 `placeId`가 존재하는지 확인 (검색 먼저 수행되어야 함)
- `category` Enum: `ATTRACTION`, `FOOD`, `ACCOMMODATION`, `ETC`
- 방 참여자만 보관함 추가 가능

---

## DoD

- [ ] 방 참여자가 장소를 보관함에 추가할 수 있다
- [ ] 동일 장소 중복 추가 시 `409` 반환된다
