# FE-005-02: 보관함 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `bookmarks`, `places`

## 개요

방의 보관함 목록을 조회한다. 카테고리 필터 지원.

---

## API 명세

### `GET /rooms/{roomId}/bookmarks?category={category}`

**Header**: `Authorization: Bearer {accessToken}`

**Query Params**
- `category` (optional): `ATTRACTION` | `FOOD` | `ACCOMMODATION` | `ETC`

**Response `200`**
```json
[
  {
    "bookmarkId": 10,
    "placeId": "ChIJ...",
    "placeName": "성산일출봉",
    "address": "제주특별자치도 서귀포시 성산읍",
    "category": "ATTRACTION",
    "memo": "꼭 가고 싶은 곳",
    "addedBy": { "userId": 1, "name": "홍길동" }
  }
]
```

---

## 구현 노트

- `bookmarks` JOIN `places` 쿼리, N+1 방지
- `category` 파라미터 없으면 전체 반환
- 추가된 순서(`bookmarks.created_at ASC`)로 정렬
