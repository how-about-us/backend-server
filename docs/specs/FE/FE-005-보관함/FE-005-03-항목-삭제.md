- Notion ID: TBD

## 개요

보관함에서 특정 장소를 제거한다.

---

## API 명세

### `DELETE /rooms/{roomId}/bookmarks/{bookmarkId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

**Error Cases**
- `403` — 방 비참여자
- `404` — 존재하지 않는 bookmark

---

## 구현 노트

- 방 참여자면 누구나 삭제 가능 (본인 추가분만? 전원 가능? — 정책 결정 필요, 우선 전원 가능으로 구현)
- `bookmarkId`가 해당 `roomId`에 속하는지 검증 (다른 방의 bookmark 삭제 방지)
