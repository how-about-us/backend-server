# FE-005-04: 보관함 메모 수정

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P3
- Notion ID: TBD
- 관련 테이블: `bookmarks`

## 개요

보관함 항목의 메모를 편집한다.

---

## API 명세

### `PATCH /rooms/{roomId}/bookmarks/{bookmarkId}`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "memo": "수정된 메모"
}
```

**Response `200`**
```json
{
  "bookmarkId": 10,
  "memo": "수정된 메모"
}
```

---

## 구현 노트

- 메모 빈 문자열 허용 (메모 삭제 = 빈 문자열로 업데이트)
- `bookmarkId`가 해당 `roomId`에 속하는지 검증

---

- [ ] 빈 문자열로 메모를 삭제할 수 있다
