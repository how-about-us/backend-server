# FE-005-05: 보관함 카테고리 변경

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P3
- Notion ID: TBD
- 관련 테이블: `bookmarks`

## 개요

보관함 항목의 카테고리를 변경한다.

---

## API 명세

### `PATCH /rooms/{roomId}/bookmarks/{bookmarkId}`

> FE-005-04 메모 수정과 동일 엔드포인트. 변경할 필드만 전송.

**Request Body**
```json
{
  "category": "FOOD"
}
```

**Response `200`**
```json
{
  "bookmarkId": 10,
  "category": "FOOD"
}
```

**Error Cases**
- `400` — 유효하지 않은 category 값

---

## 구현 노트

- 카테고리 Enum: `ATTRACTION`, `FOOD`, `ACCOMMODATION`, `ETC`
- FE-005-04와 동일 PATCH 엔드포인트에서 처리 (memo, category 동시 변경 가능)

---

## DoD

- [ ] 카테고리가 정상 변경된다
- [ ] 유효하지 않은 카테고리 값 입력 시 `400` 반환된다
