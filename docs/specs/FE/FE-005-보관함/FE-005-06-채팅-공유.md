# FE-005-06: 장소 채팅에 공유

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `bookmarks`, `places`, `messages`

## 개요

보관함 항목 또는 검색 결과 장소를 채팅 메시지로 전송한다. (`message_type: PLACE_SHARE`)

---

## API 명세

WebSocket(STOMP) 메시지로 전송. HTTP가 아님.

**STOMP SEND** `/app/rooms/{roomId}/messages`
```json
{
  "type": "PLACE_SHARE",
  "placeId": "ChIJ..."
}
```

**STOMP BROADCAST** `/topic/rooms/{roomId}`
```json
{
  "messageId": 101,
  "type": "PLACE_SHARE",
  "sender": { "userId": 1, "name": "홍길동" },
  "place": {
    "placeId": "ChIJ...",
    "name": "성산일출봉",
    "address": "...",
    "photoUrl": "https://..."
  },
  "createdAt": "2025-08-01T10:00:00"
}
```

---

## 구현 노트

- `messages` 테이블에 `message_type = PLACE_SHARE`, `place_id = :placeId`로 저장
- 채팅 히스토리 조회 시 `PLACE_SHARE` 타입은 `places` JOIN해서 장소 정보 포함 반환
- `placeId`가 `places` 테이블에 존재하지 않으면 에러
