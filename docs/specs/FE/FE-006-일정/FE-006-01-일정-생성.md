# FE-006-01: 일정 생성

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `schedules`

## 개요

방에 N일차 일정(날짜)을 추가한다.

---

## API 명세

### `POST /rooms/{roomId}/schedules`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "dayNumber": 1,
  "date": "2025-08-01"
}
```

**Response `201`**
```json
{
  "scheduleId": 1,
  "dayNumber": 1,
  "date": "2025-08-01"
}
```

**Error Cases**
- `409` — 동일 `dayNumber` 또는 `date`가 이미 존재

---

## 구현 노트

- `day_number` + `date` 두 컬럼 모두 저장 (결정됨)
- 방의 `startDate` ~ `endDate` 범위 이내인지 검증
- 방 참여자면 누구나 일정 생성 가능
