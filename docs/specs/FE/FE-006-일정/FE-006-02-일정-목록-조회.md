# FE-006-02: 일정 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `schedules`

## 개요

방의 전체 일자별 일정 목록을 반환한다.

---

## API 명세

### `GET /rooms/{roomId}/schedules`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
[
  {
    "scheduleId": 1,
    "dayNumber": 1,
    "date": "2025-08-01",
    "itemCount": 3
  },
  {
    "scheduleId": 2,
    "dayNumber": 2,
    "date": "2025-08-02",
    "itemCount": 0
  }
]
```

---

## 구현 노트

- `day_number ASC` 정렬
- `itemCount`는 서브쿼리 또는 LEFT JOIN count
