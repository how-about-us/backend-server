# FE-006-03: 일정 삭제

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `schedules`, `schedule_items`

## 개요

특정 일자 일정을 삭제한다. 해당 일자의 모든 `schedule_items`도 함께 삭제.

---

## API 명세

### `DELETE /rooms/{roomId}/schedules/{scheduleId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

---

## 구현 노트

- DB-level CASCADE: `schedule_items.schedule_id FK` → `schedules.id` ON DELETE CASCADE
- `scheduleId`가 해당 `roomId`에 속하는지 검증
