# FE-006-04: 일정에 장소 추가

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `schedule_items`, `places`

## 개요

특정 일자 일정에 장소를 추가한다. 보관함에서 또는 검색 결과에서 바로 추가 가능.

---

## API 명세

### `POST /rooms/{roomId}/schedules/{scheduleId}/items`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "placeId": "ChIJ...",
  "memo": "점심 식사 예정"
}
```

**Response `201`**
```json
{
  "itemId": 5,
  "placeId": "ChIJ...",
  "placeName": "성산일출봉",
  "orderIndex": 3,
  "memo": "점심 식사 예정"
}
```

---

## 구현 노트

- `order_index`: 현재 해당 일정의 최대 `order_index + 1`로 자동 설정
- `places` 테이블에 해당 `placeId` 존재 여부 확인
- 방 참여자면 누구나 추가 가능
