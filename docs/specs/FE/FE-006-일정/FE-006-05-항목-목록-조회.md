# FE-006-05: 일정 항목 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `schedule_items`, `places`

## 개요

특정 일자의 장소 목록을 `order_index` 순으로 반환한다.

---

## API 명세

### `GET /rooms/{roomId}/schedules/{scheduleId}/items`

**Header**: `Authorization: Bearer {accessToken}`

**Response `200`**
```json
[
  {
    "itemId": 5,
    "orderIndex": 1,
    "placeId": "ChIJ...",
    "placeName": "성산일출봉",
    "address": "...",
    "memo": "오전 방문",
    "startTime": "09:00",
    "durationMinutes": 120,
    "travelFromPrev": {
      "distanceMeters": 5000,
      "durationSeconds": 600,
      "travelMode": "DRIVING"
    }
  }
]
```

---

## 구현 노트

- `schedule_items` JOIN `places` 쿼리, N+1 방지
- `travelFromPrev`는 이전 항목과의 이동 정보 (첫 항목은 null)
- `order_index ASC` 정렬

---

## DoD

- [ ] 항목이 `order_index` 순으로 반환된다
- [ ] 이동 정보가 포함되어 반환된다
