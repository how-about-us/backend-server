# FE-006-06: 일정 항목 삭제

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `schedule_items`

## 개요

일정에서 특정 장소를 제거한다.

---

## API 명세

### `DELETE /rooms/{roomId}/schedules/{scheduleId}/items/{itemId}`

**Header**: `Authorization: Bearer {accessToken}`

**Response `204`** — No Content

---

## 구현 노트

- `itemId`가 해당 `scheduleId`에 속하는지 검증
- 삭제 후 이동 정보 재계산 트리거 (삭제된 항목의 전·후 항목 간 이동 정보 비동기 갱신)
- `order_index`는 gap 방식으로 유지 (재정렬 불필요)

- [ ] 삭제 후 전후 항목의 이동 정보가 재계산된다
