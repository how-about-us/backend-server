# FE-006-07: 일정 순서 변경 (D&D)

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `schedule_items`

## 개요

드래그 앤 드롭으로 일정 항목 순서를 변경한다. 변경 후 WebSocket으로 브로드캐스트하고, 이동 정보를 비동기로 재계산한다.

---

## API 명세

### `PATCH /rooms/{roomId}/schedules/{scheduleId}/items/reorder`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "itemOrders": [
    { "itemId": 5, "orderIndex": 1 },
    { "itemId": 3, "orderIndex": 2 },
    { "itemId": 7, "orderIndex": 3 }
  ]
}
```

**Response `200`** — 변경된 순서 그대로 반환

---

## 구현 노트

- 전체 `itemOrders` 배열을 받아 batch update (루프 N번 UPDATE 대신 bulk update)
- HTTP 응답 후 WS `/topic/rooms/{roomId}/schedule` 이벤트 발행
- `@Async`로 Google Routes API 호출 → `distance_meters`, `duration_seconds` DB 저장 → WS 재발행
