- Notion ID: TBD

## 개요

일정 항목의 방문 메모를 편집한다.

---

## API 명세

### `PATCH /rooms/{roomId}/schedules/{scheduleId}/items/{itemId}`

**Header**: `Authorization: Bearer {accessToken}`

**Request Body**
```json
{
  "memo": "수정된 메모"
}
```

**Response `200`**
```json
{
  "itemId": 5,
  "memo": "수정된 메모"
}
```
