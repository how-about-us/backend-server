# FE-006-09: 시간 설정

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `schedule_items`

## 개요

일정 항목의 방문 시작 시간과 체류 시간을 설정한다.

---

## API 명세

### `PATCH /rooms/{roomId}/schedules/{scheduleId}/items/{itemId}`

> FE-006-08 메모 수정과 동일 엔드포인트.

**Request Body**
```json
{
  "startTime": "09:00",
  "durationMinutes": 120
}
```

**Response `200`**
```json
{
  "itemId": 5,
  "startTime": "09:00",
  "durationMinutes": 120
}
```

---

## 구현 노트

- `startTime`: `HH:mm` 형식, null 허용 (미설정)
- `durationMinutes`: 양수, null 허용
- 시간 설정 후 이동 정보 재계산은 불필요 (이동 정보는 장소 좌표 기준)

---

## DoD

- [ ] 시작 시간과 체류 시간이 정상 저장된다
- [ ] null로 설정해 초기화할 수 있다
