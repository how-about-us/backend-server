# FE-007-01: 메시지 전송 (WS)

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `messages`

## 개요

WebSocket(STOMP)으로 실시간 텍스트 메시지를 전송한다.

---

## API 명세

**STOMP SEND** `/app/rooms/{roomId}/messages`
```json
{
  "type": "TEXT",
  "content": "안녕하세요!"
}
```

**STOMP BROADCAST** `/topic/rooms/{roomId}`
```json
{
  "messageId": 100,
  "type": "TEXT",
  "content": "안녕하세요!",
  "sender": { "userId": 1, "name": "홍길동", "profileImage": "https://..." },
  "createdAt": "2025-08-01T10:00:00"
}
```

---

## 구현 노트

- STOMP 연결 시 JWT 검증 (`Authorization` 헤더 또는 `connect` 프레임 헤더)
- `messages` 테이블에 저장 후 브로드캐스트
- 방 참여자가 아닌 경우 STOMP 구독 차단
