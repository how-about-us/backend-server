# FE-007-02: 메시지 목록 조회

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P1
- Notion ID: TBD
- 관련 테이블: `messages`, `places`

## 개요

방 채팅 히스토리를 페이지네이션으로 조회한다.

---

## API 명세

### `GET /rooms/{roomId}/messages?cursor={lastMessageId}&size=50`

**Header**: `Authorization: Bearer {accessToken}`

**Query Params**
- `cursor` (optional): 마지막으로 받은 `messageId`. 없으면 최신 메시지부터.
- `size`: 조회 건수 (기본 50)

**Response `200`**
```json
{
  "messages": [
    {
      "messageId": 100,
      "type": "TEXT",
      "content": "안녕하세요!",
      "sender": { "userId": 1, "name": "홍길동" },
      "createdAt": "2025-08-01T10:00:00"
    }
  ],
  "hasMore": true,
  "nextCursor": 50
}
```

---

## 구현 노트

- Cursor 기반 페이지네이션 (`WHERE id < :cursor ORDER BY id DESC LIMIT :size`)
- `PLACE_SHARE` 타입은 `places` JOIN해서 장소 정보 포함
- 방 참여자 권한 검증 필요
