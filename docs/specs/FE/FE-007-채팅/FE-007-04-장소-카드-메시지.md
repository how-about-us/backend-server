# FE-007-04: 장소 카드 메시지 전송

- 상태: 대기

- Notion ID: TBD

## 개요

장소 정보를 카드 형태로 채팅에 공유한다. FE-005-06과 동일 흐름이나 직접 검색 결과에서도 전송 가능.

> FE-005-06 (보관함 → 채팅 공유) 참고. 해당 spec의 WS 명세와 동일.

## 구현 노트

- `messages.message_type = PLACE_SHARE`
- `messages.place_id` FK → `places.google_place_id`
- 채팅 히스토리 조회 시 `PLACE_SHARE` 타입은 `places` JOIN해서 반환
