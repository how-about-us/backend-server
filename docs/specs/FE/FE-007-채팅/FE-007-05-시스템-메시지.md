# FE-007-05: 시스템 메시지

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P2
- Notion ID: TBD
- 관련 테이블: `messages`

## 개요

멤버 입퇴장 등 시스템 이벤트를 채팅에 자동으로 표시한다. (`message_type: SYSTEM`)

---

## 이벤트 종류

| 이벤트 | 메시지 예시 |
|--------|------------|
| 멤버 입장 | "홍길동 님이 입장하셨습니다." |
| 멤버 퇴장 (나가기) | "홍길동 님이 퇴장하셨습니다." |
| 멤버 강퇴 | "홍길동 님이 내보내졌습니다." |

---

## WS 브로드캐스트

**STOMP BROADCAST** `/topic/rooms/{roomId}`
```json
{
  "messageId": 101,
  "type": "SYSTEM",
  "content": "홍길동 님이 입장하셨습니다.",
  "sender": null,
  "createdAt": "2025-08-01T10:05:00"
}
```

---

## 구현 노트

- `sender_id = NULL`, `message_type = SYSTEM`으로 저장
- 멤버 입장/퇴장/강퇴 서비스 로직에서 시스템 메시지 저장 + WS 발행 호출
- 히스토리 조회 시 `SYSTEM` 타입도 포함해 반환
