# FE-008-01: AI 호출 (룰 기반 트리거)

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P3
- Notion ID: TBD
- 관련 테이블: `messages` (또는 `ai_responses` — 미결)

## 개요

채팅창의 AI 호출 버튼으로 AI에게 질의한다. AI는 방 일정·보관함 컨텍스트를 기반으로 응답.

---

## API 명세 (안)

**STOMP SEND** `/app/rooms/{roomId}/ai`
```json
{
  "query": "2일차 점심 맛집 추천해줘"
}
```

> 또는 HTTP `POST /rooms/{roomId}/ai/query` 후 WS로 응답 스트리밍 고려

---

## 구현 노트

- AI 응답 저장 방식 미결: `messages.message_type=AI_RESPONSE` vs `ai_responses` 별도 테이블 → 결정 전까지 구현 보류
- 컨텍스트 전달 범위 미결: 최소한 방 제목·여행지·날짜는 포함
- AI 응답은 `sender_id=NULL`로 채팅에 표시 (FE-008-02 참고)

- [ ] 방 메타 정보가 컨텍스트에 포함된다
