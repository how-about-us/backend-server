# FE-008-02: AI 응답 채팅에 표시

## 메타
- 상태: 대기
- 담당: backend/박주영
- 우선순위: P3
- Notion ID: TBD
- 관련 테이블: `messages` (또는 `ai_responses` — 미결)

## 개요

AI 응답을 `sender_id=NULL` 메시지로 채팅에 표시한다.

---

## WS 브로드캐스트

**STOMP BROADCAST** `/topic/rooms/{roomId}`
```json
{
  "messageId": 110,
  "type": "AI_RESPONSE",
  "content": "2일차 점심으로 제주 흑돼지 거리를 추천드려요...",
  "sender": null,
  "createdAt": "2025-08-01T10:10:00"
}
```

---

## 구현 노트

- AI 응답 완료 후 DB 저장 → WS 브로드캐스트 (스트리밍은 MVP 이후 고려)
- `sender_id = NULL`, `message_type = AI_RESPONSE`
- 채팅 히스토리 조회 시 `AI_RESPONSE` 타입도 포함
